Pour moi, le wireframing est l'étape la plus intéressante au sein d'un projet Web. Les wireframes structurent, hiérarchisent et orientent le contenu afin de guider l'ensemble des intervenants du projet.

Certaines personnes préféreront débuter cette phase de conception en réalisant les premiers croquis à main levée. Le wireframing papier permet d’obtenir un aperçu base définition de l’application avant de convertir le tout en version numérique.

Réaliser les croquis d’une application Web sans les outils adéquats n’est pas évident. Au fil de mes projets, j’ai archivé quelques liens utiles concernant le wireframing papier. Les voilà réunis dans cet article.
