Ecrit par @MagDuWebdesign • Mardi 8 Janvier 2013

Ecrit par

@MagDuWebdesign

Mardi 8 Janvier 2013

Vous cherchez des talents ?

Des centaines de designers et développeurs utilisent le Magazine du Webdesign chaque jour. Si vous êtes à la recherche de talents, c'est une bonne place pour en trouver :-)

DECOUVREZ COMMENT

Chaque semaine, un condensé d'articles, de ressources et diverses trouvailles glanées aux quatre coins des Internets.

Au sommaire :

  • Desicônes inspirées en SVG,
  • Un plugin Illustrator,
  • Des ressources jQuery,
  • Un menu off-canvas responsive.

Si vous souhaitez recevoir quotidiennement votre part du butin, n’hésitez pas à nous suivre sur Twitter et sur Facebook ou à vous abonner à notre flux RSS.

Plugin Illustrator Ai to Canvas

Le plugin Ai to Canvas autorise l’export de vecteurs et de bitmaps en éléments HTML canvas à partir d’Adobe Illustrator. Le plugin permet de rendre interactif les canvas exportés.

Plugin illustrator Ai to Canvas

Plugin jQuery Any list scroller

Tranformez n’importe quel tag <li> en liste « scrollable » avec le plugin jQuery Any list scroller.

Any list scroller

Plugin jQuery jGravity

Rendez votre site plus vivant en y ajoutant un peu de gravité avec jGravity.

Plugin jQuery jGravity

A!ertifyjs

Alertify.js est un plugin permettant de mettre en place rapidement un système de notifications JavaScript.

Alertifyjs – systeme de notifications JavaScript customisable

Animate.css

Retrouvez sur Animate.css un concentré d’exemples d’animations en CSS.

Animate.css

Redsgned

Vous passez des heures sur vos sites Web préférés. Il vous est sûrement déjà arrivé de vous dire « Ah… je n’aurais pas fait comme ça » pour une fonction ou un élément d’UI de votre site fétiche. Sur Redsgned, vous découvrirez que de nombreuses personnes pensent comme vous.

Redsgned

Plugin jQuery SocialCount

SocialCount vous apprend la vie en matière d’intégration des plugins de partages sociaux.

Plugin jQuery SocialCount

Game icons

Si vous êtes sur le point de sortir la grosse appli du moment, faites un tour sur game-icons.net avant le lancement. Vous y trouverez certainement un ou deuxicônes en SVG parfaites pour votre projet.

969 icones SVG et PNG gratuites

Ensemble d’icônes Chunky Pika

Une autre ressource d’icônes proposée par Smashing Magazine.

Smashing icons dutchicon

Menu off-canvas responsive

Un menu off-canvas responsive prêt à l’emploi. Pour plus d’informations, consultez notre article sur le concept de navigation off-canvas responsive.

Menu Off-Canvas Responsive
