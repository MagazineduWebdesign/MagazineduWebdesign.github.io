La migration vers un template responsive implique une multitude de facteurs à prendre en compte pour déterminer son choix. Quels contenus diffuser sur quels supports ?, Quelles sont les performances souhaitées ?, etc. Posez-vous les bonnes questions avant de vous lancer dans l’aventure responsive. En attendant votre introspective, vous pouvez consulter la sélection de thèmes responsive gratuits du mois de novembre.

Thème Pilot Fish

Pilot fish est un élégantthème WordPress responsive proposé dans un design minimaliste. Il dispose d’un post type dédié à l’affichage de vos créations en homepage.

Thème Leaf

Leaf a été conçu spécifiquement pour les magazines en ligne, les sites de news et les blogs. Doté d’un design épuré et moderne, Leaf s’adaptera à tous les supports grâce à ses templates responsives.

Thème ShutterLoop

ShutterLoop est un thème WordPress responsive dédié à la photographie. Gratuit, ce thème dispose d’un panel d’options complet.
