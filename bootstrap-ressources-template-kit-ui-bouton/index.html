Découvrez une sélection d'excellentes ressources Bootstrap gratuites et premium.

Bootstrap est un projet de framework développé par Twitter. Dans un premier temps, il s’agissait d’un outil de prototypage destiné à l’interne. Le projet a ensuite été mis à disposition du public sous licence Apache. La version 2.2.1 est sortie il y a quelques semaines.

Il s’agit d’un outil, donc avec quelques contraintes, notamment graphiques. Pour aller plus loin avec Bootstrap, en matière de fonctionnalités ou d’habillages graphiques, de nombreuses ressources Bootstrapées  (thèmes, kits UI et CSS) sont à votre disposition.
